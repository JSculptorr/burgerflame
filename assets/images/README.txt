Папка для собственных изображений товаров.

По умолчанию все картинки товаров загружаются с Unsplash CDN
(см. frontend/js/data.js — поле image у каждого товара).

Чтобы использовать свои изображения:
1) Положите файлы сюда (например: burger1.jpg)
2) В data.js укажите путь: image: 'assets/images/burger1.jpg'
