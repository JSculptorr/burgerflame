# 🔥 BURGER FLAME

**Премиальный сайт fast-food кафе с полным функционалом и интеграцией Telegram-бота.**

Современный коммерческий проект уровня лучших food-сервисов: dark premium дизайн с огненными акцентами, glassmorphism, плавные анимации GSAP/AOS, рабочая корзина, оформление заказа, автоматическая отправка заказов в Telegram.

## 🚀 Быстрый запуск

### Требования
- **Node.js 18+** ([скачать](https://nodejs.org/))

### Установка

```bash
# 1. Перейдите в папку backend
cd burgerflame/backend

# 2. Установите зависимости
npm install

# 3. Запустите сервер
npm start
```

Откройте в браузере: **http://localhost:3000**

✅ Сайт сразу работает. Корзина, поиск, фильтры, формы — всё функционально.
⚠️ Для отправки заказов в Telegram нужно настроить бота (см. ниже).

## 🤖 Настройка Telegram-бота

### Шаг 1. Создайте бота
1. Откройте Telegram и найдите **@BotFather**
2. Отправьте `/newbot`
3. Придумайте имя (`BurgerFlame Orders`) и username (`burgerflame_orders_bot`)
4. Скопируйте присланный **TOKEN** вида:
   ```
   123456789:AAH-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```

### Шаг 2. Получите CHAT_ID

**Вариант А — личные сообщения** (для одного владельца):
1. Найдите **@userinfobot** в Telegram → отправьте `/start`
2. Он пришлёт ваш ID (например `123456789`) — это **CHAT_ID**
3. ⚠️ Напишите своему новому боту любое сообщение (например `/start`), иначе он не сможет вам писать

**Вариант Б — группа** (для команды кафе):
1. Создайте группу в Telegram, добавьте туда бота как администратора
2. Отправьте в группе любое сообщение
3. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
4. Найдите в ответе `"chat":{"id":-1001234567890,...}` — это **CHAT_ID** (для групп со знаком минус)

### Шаг 3. Вставьте токен и ID

**Способ 1** — через `backend/config.js`:
```javascript
module.exports = {
    BOT_TOKEN: '123456789:AAH-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
    CHAT_ID:   '123456789',
    PORT:      3000
};
```

**Способ 2** — через `.env` файл (рекомендуется):
```bash
cp .env.example .env
```
И отредактируйте:
```
TELEGRAM_BOT_TOKEN=123456789:AAH-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TELEGRAM_CHAT_ID=123456789
```

### Шаг 4. Перезапустите сервер
```bash
npm start
```

В консоли должно быть: `▶ Telegram: ✅ настроен`

## 🎨 Кастомизация

### Добавить/изменить товары
Откройте `frontend/js/data.js` — массив `PRODUCTS`. Каждый товар:
```javascript
{
    id: 'b5',                    // уникальный ID
    category: 'burgers',         // burgers | pizza | doner | combo | drinks | dessert
    name: 'НАЗВАНИЕ',
    desc: 'Описание состава',
    price: 590,
    oldPrice: 790,               // необязательно (для зачёркнутой цены)
    image: 'https://...',        // URL картинки
    badges: ['hot', 'sale'],     // hot | new | sale | top
    rating: 4.9,
    bestseller: true             // показать в карусели
}
```

### Изменить цвета темы
Откройте `frontend/css/style.css` → блок `:root`. Главные переменные:
```css
--accent:    #ff6b1a;   /* основной оранжевый */
--accent-2:  #ff2d55;   /* красный */
--bg:        #0a0a0a;   /* фон */
```

### Изменить контакты, адрес, карту
- Текст контактов: `frontend/index.html` → секция `<section id="contacts">`
- Google Maps: замените `<iframe src="...">` на свою ссылку из Google Maps → "Поделиться" → "Встраивание карты"
- Соцсети, телефон, email: ищите в `<footer>` и в шапке

### Изменить стоимость доставки
`frontend/js/cart.js` → константы в начале:
```javascript
const FREE_DELIVERY_FROM = 500;  // от какой суммы бесплатно
const DELIVERY_PRICE     = 150;  // цена доставки
```

## 🛠️ API

### `POST /api/order`
Принимает JSON заказа, отправляет в Telegram.

**Тело запроса:**
```json
{
  "customer": {
    "name": "Иван",
    "phone": "+7 999 123 45 67",
    "address": "ул. Тверская, 10",
    "comment": "",
    "payment": "Картой при получении"
  },
  "items": [
    { "id": "b1", "name": "FLAME XL", "price": 590, "qty": 2 }
  ],
  "subtotal": 1180,
  "delivery": 0,
  "total": 1180,
  "createdAt": "2026-05-08T14:32:00.000Z"
}
```

**Ответ:** `{ "ok": true, "message": "Заказ отправлен в Telegram" }`

### `GET /api/health`
Проверка статуса сервера и настройки Telegram.

## 🐛 Решение проблем

| Проблема | Решение |
|----------|---------|
| `Telegram API error: chat not found` | Напишите боту `/start` в личке, или добавьте бота в группу |
| `Telegram API error: Unauthorized` | Неверный `BOT_TOKEN` — перепроверьте у @BotFather |
| Сайт открывается, но заказ не уходит | Проверьте консоль сервера и `/api/health` |
| Картинки не грузятся | Нужен интернет — изображения с Unsplash CDN |
| `npm install` ругается | Установите Node.js 18+ |
